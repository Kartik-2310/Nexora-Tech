# Nexora Tech Mobile Suite

A premium mobile experience for a multiservice IT company with three roles: Client, Admin, and Team Member. The concept focuses on a modern, scalable UI foundation with dark/light themes, premium visual hierarchy, and mobile-friendly layouts for Android and iOS.

## Highlights

- Client marketplace, project tracking, secure chat, payment, and appointment booking journeys.
- Admin command center for client management, revenue analytics, project allocation, and notifications.
- Team member hub for task updates, time tracking, and internal collaboration.
- Built with a scalable, modular UI structure ready for backend integrations.

## Getting Started

```bash
npm install
npm run start
```
